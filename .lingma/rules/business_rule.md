# Momentum 业务规则

本项目是仿照 Linear 产品实现的项目管理工具，因此需要遵循 Linear 的核心设计理念和业务规则。

## 核心理念

1. **现代产品团队工具**
   - 专为现代软件开发团队设计
   - 注重快速执行和工艺质量
   - 保持世界级产品团队的实践和原则

2. **速度与效率优先**
   - 优化工作流程以提高执行速度
   - 简化操作步骤，减少用户完成任务的时间
   - 界面设计简洁直观，减少认知负担

3. **专注与集中**
   - 帮助团队保持专注，避免干扰
   - 提供清晰的视觉层次和信息架构
   - 突出最重要的任务和信息

## 核心业务模块

### 1. 问题跟踪 (Issue Tracking)

1. **快速创建**
   - 支持快捷键创建问题 (如按 C 键)
   - 简化的新建表单，快速填写核心信息
   - 支持从任何页面快速创建问题
   - 支持富文本编辑器
   - AI自动建议功能

2. **问题属性**
   - 标题、描述 (支持 Markdown 和富文本)
   - 状态 (Backlog, Todo, In Progress, In Review, Done)
   - 优先级 (Urgent, High, Medium, Low, No priority)
   - 负责人 (Assignee)
   - 项目关联
   - 周期 (Cycle) 关联
   - 标签和标签组
   - 估算 (Estimates)
   - 截止日期
   - SLA 状态

3. **问题交互**
   - 内联评论和讨论（支持线程化评论）
   - 活动历史记录
   - 标签和自定义字段
   - 子问题支持
   - 问题关系（阻塞/被阻塞）
   - 相似问题检测

4. **视图和过滤**
   - 列表视图和看板视图切换
   - 自定义过滤器和操作符
   - 分组和排序功能
   - 泳道视图（Swimlanes）
   - 多层视图展示

### 2. 项目管理 (Project Management)

1. **项目结构**
   - 项目概览页面显示进度和关键指标
   - 项目文档和规范集中管理
   - 里程碑和交付物跟踪
   - 项目依赖关系管理
   - 项目模板支持

2. **项目状态**
   - 可自定义的项目状态（如 Idea, Accepted, In Progress, Shipped, Canceled）
   - 项目健康度检查
   - 进度可视化展示
   - 项目更新（Project Updates）

3. **项目协作**
   - 协作编辑项目文档
   - 内联评论和反馈
   - 文本到问题命令（Text-to-issue commands）
   - 项目属性管理

4. **高级功能**
   - 战略性项目计划（Initiatives）
   - 跨团队项目协作
   - 项目标签和标签组
   - 项目健康状态跟踪

### 3. 周期管理 (Cycle Management)

1. **工作节奏**
   - 固定时间周期 (如2周一个周期)
   - 周期内任务范围限定
   - 周期目标设定和跟踪
   - 自动化周期计划

2. **周期功能**
   - 未完成工作自动滚入下一周期
   - 活跃问题自动添加到当前周期
   - 可配置的开始日期、持续时间和冷却期
   - 周期进度洞察

3. **进度可视化**
   - 燃起图显示完成进度
   - 工作量分布统计
   - 团队成员贡献度
   - 范围蔓延监控和延迟预测

### 4. 路线图 (Roadmap)

1. **战略规划**
   - 高层次产品规划视图
   - 时间轴布局展示项目排期
   - 项目依赖关系可视化
   - 多层级视图（产品管道、季度规划、时间线）

2. **进度跟踪**
   - 项目状态实时更新
   - 关键里程碑标记
   - 风险和阻塞因素标识
   - 项目路线图共享

3. **可视化选项**
   - 列表、看板、时间线视图
   - 泳道支持
   - 动态规划功能

### 5. 团队协作 (Team Collaboration)

1. **收件箱 (Inbox)**
   - 所有新问题和通知的集中入口
   - 支持筛选和搜索
   - 未读标记和处理状态
   - 通知优先级管理（响应重要问题，推迟其他问题）

2. **个人工作台 (My Issues)**
   - 个人负责问题的集中展示
   - 工作负载可视化
   - 个人进度跟踪

3. **待处理管理 (Triage)**
   - 共享团队收件箱管理传入的bug报告、功能请求等
   - 问题审查、优先级排序和分配
   - 自动分配 incoming issues
   - SLA（服务等级协议）支持

### 6. 洞察和报告 (Insights & Reporting)

1. **实时分析**
   - 任何工作流的即时报告
   - 数据聚合、分段和可视化
   - 趋势识别和阻塞因素消除

2. **仪表板**
   - 为每个团队构建自定义仪表板
   - 灵活的布局选项
   - 可共享和隐私控制

3. **度量指标**
   - 问题数量、工作量、周期时间
   - 分拣时间、前置时间、问题年龄
   - 按状态、负责人、优先级等分段
   - 时间序列分组（日、周、月、季度、年）

### 7. 集成和自动化 (Integrations & Automations)

1. **第三方集成**
   - Git自动化（GitHub、GitLab）
   - 客户支持工具集成（Zendesk、Front、Intercom）
   - Slack集成
   - Google Sheets、Fivetran等数据集成

2. **自动化工作流**
   - PR/MR状态自动更新问题状态
   - Magic words支持（如"close"、"fixes"等）
   - 自动分配问题和更新状态
   - PR审查状态查看
   - 基于特定分支的自定义规则

3. **数据管理**
   - 自动关闭和归档功能
   - CSV导出
   - 数据仓库同步

   - 一致的设计语言和交互模式
   - 暗色主题为主，减少视觉疲劳

2. **高效操作**
   - 键盘快捷键支持
   - 批量操作功能
   - 拖拽交互支持
   - 键盘优先设计

3. **实时同步**
   - 多用户协作实时更新
   - 数据变更即时同步
   - 状态变化及时通知

4. **智能辅助**
   - AI驱动的相似问题检测
   - 自动建议和智能填充
   - 自动化工作流减少手动操作

## 工作流程规范

### 问题生命周期

1. **创建**: 问题被创建，初始状态为 Backlog
2. **规划**: 问题被评审并分配到具体周期
3. **处理**: 开发人员开始处理问题，状态变更为 In Progress
4. **审查**: 完成开发后进入审查状态 In Review
5. **完成**: 审查通过后标记为 Done

### 项目管理流程

1. **立项**: 创建项目并定义目标
2. **规划**: 制定项目计划和里程碑
3. **执行**: 分配任务并跟踪进度
4. **交付**: 完成项目目标并总结

### 周期管理流程

1. **计划**: 确定周期目标和任务范围
2. **执行**: 团队成员处理分配的任务
3. **评审**: 周期结束时回顾完成情况
4. **改进**: 根据回顾结果优化下一周期

### 待处理管理流程

1. **接收**: 新问题进入待处理队列
2. **审查**: 团队审查问题并确定优先级
3. **分配**: 将问题分配给适当的团队成员
4. **处理**: 问题进入正常的生命周期

## 数据模型规范

### 核心实体

1. **Issue (问题)**
   - 标题、描述
   - 状态、优先级
   - 负责人、创建人
   - 项目、周期关联
   - 标签、估算
   - 创建时间、更新时间、截止日期
   - SLA状态

2. **Project (项目)**
   - 名称、描述
   - 负责人
   - 开始时间、结束时间
   - 进度状态
   - 里程碑
   - 依赖关系

3. **Cycle (周期)**
   - 名称、目标
   - 开始时间、结束时间
   - 关联问题列表
   - 进度指标

4. **Team (团队)**
   - 名称、描述
   - 成员列表
   - 关联项目
   - 待处理管理设置

5. **User (用户)**
   - 姓名、邮箱
   - 头像
   - 角色和权限

6. **Initiative (计划)**
   - 名称、描述
   - 关联项目
   - 进度状态
   - 战略目标

## 界面交互规范

1. **抽屉式详情页**
   - 从右侧滑出的问题详情面板
   - 不跳转页面，保持上下文
   - 支持快速关闭和切换

2. **列表和看板视图**
   - 支持多种视图模式切换
   - 拖拽重新排序
   - 分组和筛选功能
   - 泳道支持

3. **快捷操作**
   - 悬停显示操作按钮
   - 批量选择和操作
   - 右键菜单支持
   - 键盘快捷键

4. **智能交互**
   - 相关问题自动提示
   - AI辅助输入建议
   - 自动完成和智能填充

## 通知和沟通

1. **收件箱通知**
   - 所有相关通知集中展示
   - 未读标记和已读状态
   - 支持标记和过滤
   - 优先级管理

2. **实时更新**
   - 协作内容实时刷新
   - 状态变更即时通知
   - 评论和提及提醒

3. **外部沟通**
   - Slack集成支持
   - 客户支持工具集成
   - 项目更新分享
